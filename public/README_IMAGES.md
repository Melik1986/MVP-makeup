# Необходимые изображения

Для работы лендинга необходимо добавить следующие изображения в папку `public/`:

## Обязательные изображения:

1. **model.jpg** - Изображение модели для Hero секции
   - Рекомендуемый размер: 800x1200px
   - Формат: JPG или WebP
   - Оптимизированное для веба

2. **cosmetic-tube.png** - Косметическая туба (декоративный элемент слева)
   - Рекомендуемый размер: 300x400px
   - Формат: PNG с прозрачным фоном

3. **makeup-brush.png** - Кисть для макияжа (декоративный элемент справа)
   - Рекомендуемый размер: 300x400px
   - Формат: PNG с прозрачным фоном

4. **audience-1.jpg** - Изображение для первой карточки аудитории
   - Рекомендуемый размер: 400x400px (круглое)
   - Формат: JPG или WebP

5. **audience-2.jpg** - Изображение для второй карточки аудитории
   - Рекомендуемый размер: 400x400px (круглое)
   - Формат: JPG или WebP

6. **audience-3.jpg** - Изображение для третьей карточки аудитории
   - Рекомендуемый размер: 400x400px (круглое)
   - Формат: JPG или WebP

7. **audience-4.jpg** - Изображение для четвертой карточки аудитории
   - Рекомендуемый размер: 400x400px (круглое)
   - Формат: JPG или WebP

## Альтернатива:

Если изображения отсутствуют, можно использовать placeholder сервисы или временные изображения:

- https://placehold.co/ - для временных placeholder'ов
- https://picsum.photos/ - для случайных изображений
